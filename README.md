# Privacy Policy for Netflix IMDb Ratings

**Last Updated:** December 10, 2025

## Overview

Netflix IMDb Ratings is a browser extension that displays IMDb ratings on Netflix content. We are committed to protecting your privacy.

## Data Collection

**We DO NOT collect, store, or transmit any personal information.**

### What We Access:

- **Netflix Page Content**: The extension reads Netflix page content to identify movie and TV show titles
- **Local Storage**: Ratings are cached locally in your browser to improve performance

### What We DO NOT Collect:

- ❌ Personal information
- ❌ Browsing history
- ❌ Netflix account information
- ❌ Viewing habits or preferences
- ❌ Any identifiable user data

## Third-Party Services

### OMDb API

- The extension makes requests to OMDb API (www.omdbapi.com) to fetch IMDb ratings
- Only movie/TV show titles are sent to OMDb API
- No personal information is sent
- OMDb API has its own privacy policy: https://www.omdbapi.com/legal.htm

## Data Storage

### Local Cache:

- IMDb ratings are stored locally in your browser using Chrome's storage API
- Cache expires automatically after 7 days
- Cache is stored only on your device and is never transmitted
- You can clear the cache by removing the extension

## Permissions Explained

### Storage Permission:

- Used to cache IMDb ratings locally for faster performance
- Reduces API calls and improves user experience
- All data stays on your device

### Host Permission (omdbapi.com):

- Required to fetch IMDb ratings from OMDb API
- Only movie/TV titles are sent (no personal data)

### Content Script Permission (netflix.com):

- Allows the extension to display ratings on Netflix pages
- Reads page content to identify movies and shows
- Does not collect or transmit your viewing data

## Data Security

- All cached data is stored locally in your browser
- No data is transmitted to our servers (we don't have any servers)
- The extension operates entirely client-side
- Communication with OMDb API uses secure HTTPS connections

## Children's Privacy

This extension does not knowingly collect any information from anyone, including children under 13.

## Changes to Privacy Policy

We may update this privacy policy from time to time. Updates will be posted with a new "Last Updated" date.

## Contact

For questions about this privacy policy or the extension, please contact us via the Chrome Web Store support page.

## Your Rights

You have the right to:

- Remove the extension at any time
- Clear all cached data by removing the extension
- Review the extension's source code

## Consent

By using this extension, you consent to this privacy policy.
